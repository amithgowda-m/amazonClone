# amazonClone

Amazon Clone created for Hackathon team. 

## Project Setup

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev` (we've set a script for this) Or `node index.js`.
4. Ensure `.env` is setup properly in backend with `MONGO_URI`!

## Team Distribution
